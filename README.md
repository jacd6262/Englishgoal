## 📖 Acerca del Proyecto

**English Goal** es la herramienta definitiva para estudiantes que desean llevar un control preciso y motivador de las horas que dedican a dominar el idioma inglés. Con herramientas de temporizador integradas, gestión de metas y un historial detallado, nunca fue tan fácil medir tu progreso hacia la fluidez.

¡Lleva tu aprendizaje al siguiente nivel! 🚀

## ✨ Características Principales

- ⏱️ **Cronómetro Integrado**: Mide de forma exacta tus sesiones de escucha, lectura, escritura o habla.
- 📊 **Historial Completo**: Consulta fácilmente qué actividades realizaste y cuánto tiempo invertiste.
- 🎯 **Gestión de Metas (Goals)**: Fíjate objetivos diarios, semanales o mensuales.
- 🎨 **Diseño Moderno e Intuitivo**: Interfaz fluida basada en componentes de alta calidad (shadcn/ui + TailwindCSS).

## 🛠️ Tecnologías y Arquitectura

El proyecto está dividido en dos partes principales: **Frontend** y **Backend**.

### 💻 Frontend (Cliente)
Aplicación React ultra rápida gracias a Vite, con un estilo impecable.
- **Framework**: React 19
- **Bundler**: Vite
- **Estilos**: Tailwind CSS v4 + Animaciones personalizadas
- **UI Toolkit**: Radix UI + shadcn/ui
- **Iconos**: Lucide React
- **Peticiones HTTP**: Axios
- **Lenguaje**: TypeScript

### ⚙️ Backend (Servidor)
API robusta y rápida para gestionar los datos de los usuarios y su progreso.
- **Entorno**: Node.js
- **Framework**: Express.js
- **Base de Datos**: MySQL (Consultas eficientes con `mysql2`)
- **Seguridad y Control**: CORS implementado

## 🚀 Cómo Empezar (Instalación Local)

Si deseas clonar el proyecto y probarlo en tu máquina local, sigue estos pasos:

### 1. Requisitos Previos
- Tener instalado [Node.js](https://nodejs.org/es/).
- Tener instalado y corriendo un servidor [MySQL](https://www.mysql.com/).

### 2. Clonar el repositorio
```bash
git clone https://github.com/jacd6262/Englishgoal.git
cd Englishgoal
```

### 3. Configurar la Base de Datos y Backend
1. Entra a la carpeta del backend:
   ```bash
   cd backend
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Ejecuta el servidor en modo desarrollo:
   ```bash
   npm run dev
   ```

### 4. Configurar el Frontend
1. Abre otra terminal y entra a la carpeta del frontend:
   ```bash
   cd frontend
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Ejecuta el entorno de desarrollo:
   ```bash
   npm run dev
   ```
4. ¡Abre la URL proporcionada por Vite (usualmente `http://localhost:5173`) en tu navegador y disfruta! 🎉
